Web Note

1280px and 1920px are the two standard widths for web design. A 1280px website will look great on laptops and mobile devices but not so great on large monitors. To ensure your site looks just as good on big screens as it does on small screens, set your max site width to 1920px or more

How to Learn HTML for Free

HTMLdescribes the structure of Web pages using markup.

"h" Headline tag

"h1" to "h6" tags are used to define HTML headings.

"P" Pragraph tag

The "p" tag defines a paragraph. Browsers automatically add a single blank line before and after each "p" element.

"abbr" Word full form tag

Tag is a language that describes the style of an HTML document . Describes Word full form

"address" For uses address tag

Tag defines the contact information for the author/owner of a document or an article.

"article" For uses specifiec content or Written article tag.

The "article" tag specifies independent, self-contained content. An article should make sense on its own and it should be possible to distribute it independently from the rest of the site. Potential sources for the "article" element: Forum post Blog post News story Note: The "article" element does not render as anything special in a browser.
